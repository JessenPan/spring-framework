# Spring代码注释

本代码库是基于spring3.2.x版本的代码注释.<br/>
注释从四个角度进行描述
- spring框架功能实现角度,即通过注释阐述spring实现原理
- 设计模式的角度,通过学习和思考spring源码使用的设计模式来提高自身技能
- 设计原则的角度,通过经典的设计SOLID准则来剖析spring的设计理念
- 提炼代码tips角度,即学习spring中的一些tips或者tracks
